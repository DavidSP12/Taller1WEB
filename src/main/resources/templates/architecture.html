Diagrama de Contenedor

El diagrama de contenedor muestra los componentes de alto nivel que conforman el sistema y cómo se comunican entre sí. Detalla la arquitectura técnica y las decisiones de tecnología.

Frontend

Aplicación web desarrollada con Thymeleaf y Bootstrap que proporciona la interfaz de usuario.

Backend

API RESTful desarrollada con Spring Boot que implementa la lógica de negocio.

Base de Datos

Sistema de almacenamiento persistente que utiliza MySQL para guardar los datos del sistema.

Diagrama de Componentes

El diagrama de componentes desglosa cada contenedor en componentes más pequeños, mostrando cómo se implementa la funcionalidad dentro de cada contenedor.

  • Controladores: Gestionan las solicitudes HTTP y devuelven respuestas apropiadas.
  • Servicios: Implementan la lógica de negocio y coordinan las operaciones.
  • Repositorios: Proporcionan acceso a los datos almacenados en la base de datos.
  • Modelos: Representan las entidades de datos y las relaciones entre ellas.
